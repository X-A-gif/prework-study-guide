HTML

  • The html element represents the root (top-level element) of an HTML document.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The main element represent the dominant content of the page.
  • The section element represents a generic standalone section of a document, which doesn't have a more specific semantic element to represent it.
  • The footer element represents a footer for its nearest ancestor sectioning content or sectioning root element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Width and height set an elements width and height
  • Border creates a border around an element
  • The box-shadow property enables you to cast a drop shadow from the frame of almost any element

Git

  • git status: checks whick branch youre currently working in
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages changes made
  • git commit -m "message" commits changes made
  • git push upload local repository content to a remote repository

JavaScript

  • Control flow is the order in which a computer executes code in a script.
  • A variable is a named container that allows us to store data in our code.
  • The const declaration creates block-scoped constants, much like variables declared using the let keyword. The value of a constant can't be changed through reassignment.
  • The Array object, as with arrays in other programming languages, enables storing a collection of multiple items under a single variable name.
  • Data tpyes:
    • String: a sequence of characters that represent text
    • Number: a numeric data type
    • Boolean: a logical data type that can only be true or false
    • Null: a special keyword denoting a null value
    • Undefined: a top-level property whose value is undefined
    • Symbol: a data type whose instances are unique and immutable
  • A function in JavaScript is similar to a procedure—a set of statements that performs a task or calculates a value, but for a procedure to qualify as a function, it should take some input and return an output where there is some obvious relationship between the input and the output.
